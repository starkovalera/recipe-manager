# Responsive Review — Feedback Refinement

Status: Browser-checked at 1440 × 900, 1024 × 768, and 390 × 844  
Updated: 2026-07-22

## Wide desktop

Import Info reallocates horizontal space while leaving the recipe visible and interactive. The 460 px drawer supports readable flag text and the resource hierarchy without becoming a second full page.

## Tablet around 1024 px

Import Info overlays the unchanged recipe instead of compressing two content columns into an unusable width. The dimmed, inert background and full-height drawer make the temporary context clear.

## Mobile

Import Info becomes a bottom sheet. Resource groups retain parent/child indentation and fit without horizontal scrolling. Focus uses Ingredients / Instructions tabs and keeps View / Focus / Edit plus Import Info directly available.

## Follow-up

High fidelity should test very long source URLs, localized labels, safe-area insets, on-screen keyboards, and a 320 px minimum width.
